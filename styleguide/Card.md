Card
================
Card: ver.0.0.0 
---
**Typical render usage:**

```js
const CardExample = () => (
  <Card align="center" styled round>
    <Card.Header>
      <Card.Icon icon="star" />
      Card Header
      <Card.Icon icon="star" />
    </Card.Header>
    <Card.Content>Card Content</Card.Content>
    <Card.Footer>
      <Card.Icon icon="star" />
      Card Footer
      <Card.Icon icon="star" />
    </Card.Footer>
  </Card>
);

export default CardExample;
```
